# Reviewer Agent Skills

## Skills

### review.validate.compliance
- **Purpose**: Validate implementation compliance with specifications
- **Inputs**: Implementation code, spec.md content, acceptance criteria
- **Outputs**: Compliance validation report with identified deviations
- **Specs referenced**: @specs/<feature>/spec.md, @specs/<feature>/plan.md

### review.validate.quality
- **Purpose**: Validate code quality and adherence to standards
- **Inputs**: Implementation code, quality standards, coding guidelines
- **Outputs**: Quality assessment report with improvement recommendations
- **Specs referenced**: @specs/quality/, @specs/<feature>/plan.md

### review.validate.test.coverage
- **Purpose**: Validate test coverage and quality assurance measures
- **Inputs**: Test implementations, test requirements, coverage standards
- **Outputs**: Test coverage assessment report with gaps identified
- **Specs referenced**: @specs/<feature>/tasks.md, @specs/quality/

### review.validate.hackathon.readiness
- **Purpose**: Assess hackathon readiness and deliverable completeness
- **Inputs**: All implementation artifacts, hackathon requirements, submission criteria
- **Outputs**: Readiness assessment with identified gaps or issues
- **Specs referenced**: @specs/<feature>/spec.md, @specs/<feature>/plan.md

### review.validate.integration
- **Purpose**: Validate integrated system functionality and performance
- **Inputs**: Integrated system, performance requirements, functionality criteria
- **Outputs**: Integration validation report with identified issues
- **Specs referenced**: @specs/<feature>/spec.md, @specs/<feature>/plan.md
