# Frontend Engineer Agent Skills

## Skills

### frontend.implement.component
- **Purpose**: Implement frontend components according to design specifications
- **Inputs**: UI/UX designs, component requirements, accessibility guidelines
- **Outputs**: Component implementation files
- **Specs referenced**: @specs/<feature>/design/, @specs/<feature>/plan.md

### frontend.implement.page
- **Purpose**: Implement complete page layouts and navigation
- **Inputs**: Page requirements, navigation structure, user flow specifications
- **Outputs**: Page implementation files
- **Specs referenced**: @specs/<feature>/spec.md, @specs/<feature>/plan.md

### frontend.implement.integration
- **Purpose**: Integrate frontend with backend APIs and services
- **Inputs**: API contracts, authentication requirements, data flow specifications
- **Outputs**: API integration and service layer files
- **Specs referenced**: @specs/contracts/, @specs/<feature>/plan.md

### frontend.implement.security
- **Purpose**: Implement client-side security measures and best practices
- **Inputs**: Security requirements, authentication flows, data protection needs
- **Outputs**: Security implementation files
- **Specs referenced**: @specs/<feature>/plan.md, @specs/<feature>/spec.md

### frontend.implement.responsive
- **Purpose**: Ensure responsive design and cross-platform compatibility
- **Inputs**: Responsive design requirements, device specifications, accessibility standards
- **Outputs**: Responsive styling and layout files
- **Specs referenced**: @specs/<feature>/design/, @specs/<feature>/plan.md
