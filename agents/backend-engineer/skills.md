# Backend Engineer Agent Skills

## Skills

### backend.implement.service
- **Purpose**: Implement backend service classes with business logic
- **Inputs**: Implementation plan, data models, business requirements
- **Outputs**: Service class implementation files
- **Specs referenced**: @specs/<feature>/plan.md, @specs/<feature>/data-model.md

### backend.implement.controller
- **Purpose**: Implement backend controller classes with API endpoints
- **Inputs**: API contracts, service implementations, authentication requirements
- **Outputs**: Controller implementation files
- **Specs referenced**: @specs/contracts/, @specs/<feature>/plan.md

### backend.implement.model
- **Purpose**: Implement data models based on specifications
- **Inputs**: Data model requirements, validation rules, persistence needs
- **Outputs**: Data model implementation files
- **Specs referenced**: @specs/<feature>/data-model.md, @specs/<feature>/plan.md

### backend.implement.auth
- **Purpose**: Implement authentication and authorization mechanisms
- **Inputs**: Security requirements, authentication flows, user management needs
- **Outputs**: Authentication implementation files
- **Specs referenced**: @specs/<feature>/plan.md, @specs/<feature>/spec.md

### backend.integrate.database
- **Purpose**: Set up and integrate database connections and migrations
- **Inputs**: Data models, schema requirements, persistence needs
- **Outputs**: Database configuration and migration files
- **Specs referenced**: @specs/<feature>/data-model.md, @specs/<feature>/plan.md
